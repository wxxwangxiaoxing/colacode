# 鸡翅Club (ColaCode) 学习笔记 - Wx 微信模块

## 第六章：Wx 微信模块

### 6.1 模块概览

Wx 微信模块是 ColaCode 与微信公众号/小程序交互的桥梁，负责接收和响应微信服务器推送的消息和事件。

**核心功能：**
- 🔐 服务器验证 (SHA-1 签名校验)
- 📨 消息接收与解析 (XML)
- 🤖 自动回复 (关键词回复/欢迎语)
- 📋 事件处理 (关注/取关/菜单点击)
- 🔗 用户绑定 (OpenID ↔ UserId)

### 6.2 微信公众号通信原理

```
用户发送消息
    ↓
微信客户端
    ↓
微信服务器 (msg.weixin.qq.com)
    ↓ HTTP POST (XML)
ColaCode Wx 服务 (:3012)
    ↓ 解析 XML → 处理 → 构建响应 XML
微信服务器
    ↓
微信客户端 (展示回复)
```

**重要：** 微信服务器超时时间为 5 秒，必须在 5 秒内返回响应。

### 6.3 策略模式 - 微信消息处理

#### 6.3.1 架构设计

```
WxMessageHandler (策略接口)
    ├── getMsgType()  → 消息类型 (text, image, event...)
    ├── getEvent()    → 事件类型 (subscribe, CLICK...)
    └── handle()      → 处理消息并返回响应

具体实现:
    ├── TextMessageHandler (text:)
    ├── SubscribeEventHandler (event:subscribe)
    ├── UnsubscribeEventHandler (event:unsubscribe)
    ├── ClickEventHandler (event:CLICK)
    └── DefaultMessageHandler (兜底)
```

#### 6.3.2 Handler 注册机制

```java
@Component
public class WxMessageHandlerFactory {
    @Resource
    private List<WxMessageHandler> handlerList;

    private Map<String, WxMessageHandler> handlerMap = new HashMap<>();

    @PostConstruct
    public void register() {
        for (WxMessageHandler handler : handlerList) {
            String key = handler.getMsgType() + ":" + handler.getEvent();
            handlerMap.put(key, handler);
        }
    }

    public WxMessageHandler getHandler(String msgType, String event) {
        String key = msgType + ":" + (event != null ? event : "");
        return handlerMap.getOrDefault(key, defaultHandler);
    }
}
```

### 6.5 服务器验证

微信公众号配置时，微信会发送 GET 请求验证服务器：

```
GET /wx?signature=xxx&timestamp=xxx&nonce=xxx&echostr=xxx
```

**验证算法：**
1. 将 `token`, `timestamp`, `nonce` 三个参数字典序排序
2. 拼接成一个字符串
3. SHA-1 加密
4. 与 `signature` 比较

```java
public static boolean checkSignature(String token, String signature, String timestamp, String nonce) {
    String[] arr = new String[]{token, timestamp, nonce};
    Arrays.sort(arr); // 字典序排序

    StringBuilder content = new StringBuilder();
    for (String s : arr) {
        content.append(s);
    }

    MessageDigest md = MessageDigest.getInstance("SHA-1");
    byte[] digest = md.digest(content.toString().getBytes());
    String tmpStr = byteToHex(digest);
    return tmpStr.equals(signature);
}
```

### 6.7 XStream XML 处理

微信使用 XML 而非 JSON，使用 XStream 进行序列化/反序列化：

```java
@XStreamAlias("xml")
public class WxMessage {
    @XStreamAlias("ToUserName")
    private String toUserName;

    @XStreamAlias("FromUserName")
    private String fromUserName;

    @XStreamAlias("MsgType")
    private String msgType;

    @XStreamAlias("Content")
    private String content;
}
```

**解析 XML：**
```java
XStream xStream = new XStream(new DomDriver());
xStream.processAnnotations(WxMessage.class);
xStream.ignoreUnknownElements(); // 忽略未知字段

WxMessage msg = (WxMessage) xStream.fromXML(xmlString);
```

**生成 XML：**
```java
XStream xStream = new XStream(new DomDriver());
xStream.processAnnotations(WxTextMessage.class);
xStream.alias("xml", WxTextMessage.class);

String xml = xStream.toXML(response);
```
