## 1. 项目概述

### 1.1 产品定位

ColaCode 是一个面向程序员的综合性学习社区平台，集题库练习、AI模拟面试、社区交流于一体，帮助开发者提升技术能力、分享学习经验。

### 1.2 目标用户

| 用户角色 | 描述 | 核心需求 |
|---------|------|---------|
| **普通用户** | 注册并登录的程序员 | 刷题练习、模拟面试、社区交流 |
| **管理员** | 平台运营人员 | 用户管理、内容审核、系统配置 |
| **访客** | 未登录用户 | 浏览公开内容、了解平台功能 |

## 2. 设计规范

### 2.1 色彩系统

```

辅助色: #52C41A (成功绿)  #FAAD14 (警告黄)  #F5222D (错误红)

       #D9D9D9 (边框)    #F0F0F0 (分割线)  #F5F5F5 (背景)

```
中文: "PingFang SC", "Microsoft YaHei", sans-serif

```

```
桌面端: ≥1200px (默认)
