## 项目经验

### 项目一：ColaCode 程序员社区平台 (进行中)

**项目简介**：一个面向程序员的微服务社区平台，支持题库练习、AI 面试、社区交流等功能。

**技术架构**：
- Spring Cloud 微服务架构 (Gateway + 9 个业务服务)
- Nacos 注册/配置中心，Sentinel 限流
- MySQL + Redis + Elasticsearch 数据存储
- Sa-Token 鉴权，RocketMQ 消息队列

**负责模块**：

1. **用户认证模块 (colacode-auth)**
   - 实现用户登录、注册、登出功能
   - 集成 Sa-Token 实现 Token 鉴权
   - 实现登录风险控制（IP 限流、失败计数）
   - 密码升级机制（明文 → BCrypt）

2. **题库服务模块 (colacode-subject)**
   - 支持单选题、多选题、判断题、简答题四种题型
   - 使用策略模式处理不同题型评分逻辑
   - 集成 Elasticsearch 实现题目搜索
   - 使用 RocketMQ 异步处理点赞事件

3. **练习服务模块 (colacode-practice)**
   - 实现练习套餐、答题记录、成绩统计
   - 通过 OpenFeign 调用题库服务获取题目
   - 实现排行榜功能

4. **AI 面试模块 (colacode-ai)**
   - 集成 Spring AI 实现 AI 面试功能
   - 支持面试题生成、答案评分
   - 实现降级服务（无 API Key 时使用 Mock）

5. **网关模块 (colacode-gateway)**
   - 统一路由配置，实现负载均衡
   - Sa-Token 响应式鉴权
   - Sentinel 限流配置
   - 登录用户信息传递给下游服务

**解决的技术难点**：
- 解决了多服务间 Token 共享问题（统一 Redis database）
- 解决了 Nacos 本地开发配置问题（禁用 import-check）
- 实现了 Spring AI 无 API Key 时的服务降级
