# 社区应用原型设计 - 鸡翅Club

## 设计背景

鸡翅Club是一个面向程序员的学习社区，用户可以发布技术笔记、分享学习心得、交流问题。原型图需要展示社区的核心交互场景：动态流、详情页、消息会话。

## 页面结构

### 1. 动态流页面（首页）
- 顶部导航栏：Logo + 搜索 + 发布按钮 + 消息图标 + 我的
- 分类标签：全部、技术、职场、问答、生活
- 动态卡片流：头像、昵称、发布时间、內容、图文、点赞评论数
- 底部发布浮窗

### 2. 动态详情页
- 返回导航
- 动态内容区：完整文本、多图网格
- 评论区：评论输入框、评论列表、回复

### 3. 消息页面
- 顶部：消息tab（通知、私信）
- 消息列表：用户头像、最后一条消息、时间和未读数
- 消息详情：会话记录、输入框

## 视觉规范

卡片采用白色背景配合暖灰色调边框，圆角12px。头像统一为48px圆形。图片采用3列网格布局，圆角8px。点赞和评论图标使用线性风格，颜色为暖灰色。分隔线采用1px虚线。

## 交互说明

点击卡片跳转详情，长按图片预览。下拉刷新动态流，上拉加载更多。评论发送后自动追加到列表。消息未读时头像右上角显示小红点。
